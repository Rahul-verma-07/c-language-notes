# First C Program

## Hello World Program

```c
#include <stdio.h>

int main() {
    printf("Hello World!");
    return 0;
}
```

## Explanation

* `#include <stdio.h>` includes the standard input-output library.
* `main()` is the entry point of the program.
* `printf()` displays output on the screen.
* `return 0;` indicates successful execution.
