# File Input/Output (File I/O)

File handling in C allows data to be stored permanently in files.

Normally, data stored in variables is lost when the program terminates. Files help store data permanently.

## Advantages of File Handling

* Permanent storage of data
* Easy retrieval of information
* Efficient handling of large data
* Data sharing between programs

## Types of Files

### Text Files

Examples:

```text
notes.txt
data.txt
```

### Binary Files

Examples:

```text
student.dat
employee.dat
```
