# printf() Function

The `printf()` function is used to display output on the screen.

## Syntax

```c
printf("message");
```

## Example

```c
#include <stdio.h>

int main() {
    int a = 10;

    printf("The value of a is %d", a);

    return 0;
}
```

## New Line Character

```c
\n
```

Used to move the cursor to the next line.
