# Data Types

Data types specify the type of data a variable can store.

## Common Data Types

| Data Type | Format Specifier | Description      |
| --------- | ---------------- | ---------------- |
| int       | %d               | Integer values   |
| float     | %f               | Decimal values   |
| char      | %c               | Single character |

## Example

```c
int age = 20;
float salary = 25000.50;
char grade = 'A';
```
