# Rules for Naming Variables

A variable name must follow these rules:

1. The first character must be an alphabet or underscore (`_`).
2. Spaces are not allowed.
3. Special symbols are not allowed except underscore (`_`).
4. Variable names are case-sensitive.

## Valid Examples

```c
age
_marks
studentName
```

## Invalid Examples

```c
2age
student name
student-name
```
