# Accessing Strings

Strings can be displayed using the `%s` format specifier.

## Example

```c
#include <stdio.h>

int main() {

    char str[] = "Rahul";

    printf("%s", str);

    return 0;
}
```

## Output

```text
Rahul
```

### Format Specifier

```c
%s
```

Used to print a complete string.
