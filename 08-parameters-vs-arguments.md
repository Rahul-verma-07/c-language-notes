# Parameters vs Arguments

Parameters and arguments are often confused.

## Parameters

Parameters are variables declared in the function definition.

```c
int sum(int a, int b)
```

Here:

```c
a
b
```

are parameters.

## Arguments

Arguments are actual values passed during a function call.

```c
sum(2, 3);
```

Here:

```c
2
3
```

are arguments.

## Important Points

* Parameters act as placeholders.
* Arguments provide actual values.
* A function can return only one value at a time.
